Hotel Management System (C++) Github Repository

The Hotel Management System facilitates the hotel room booking process through making a reservation, accessing information on reserved or unreserved rooms and outputting information detailing the amenities offered by the rooms. Offered me the challenge of working with object-oriented programming concepts and layers of conditional statements. Utilizes Class Objects to store booking information. Navigating displayed menus is handled by various control flow statements. Several data types are used to store and access data.

Data Structures Repository (C++) Github Repository

This project is a collection of header, implementations and demonstrations files for various data structures encountered during my COP 3530 (Data Structures) course and beyond. Understanding data structures is fundamental to becoming a proficient programmer or computer scientist. By studying and implementing these data structures, developers can improve their problem-solving skills, optimize algorithm efficiency, and build more robust and scalable software applications. As I continue this course and genearlly grow as a Software Engineer, I will continue to cement my knowledge on Data Structures by updating this repo with more classes and more implementation files. For now I'm planning to add: Double Linked List, Stacks & Queues, Trees. Currently there is a linked list integrated.

AVR Simple Simon Game Github Repository

This AVR Assembly code project contains interrupts, timers, meticulous commenting all supporting a physical simple simon game. The arduino wiring was a challenge and although we couldn't succesfully play through a game sequence we did learn a lot about collaborating on github, debugging and conflict resolution. With the team all trying different wiring configuration and so much debate over what the most efficient design we all had to develop great communication skills if we wanted to complete our prototype and presentiong by the deadline. Overall, a great learning experince and lots of fun.
